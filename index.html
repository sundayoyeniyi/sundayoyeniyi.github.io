dotfiles

This repo holds the way I choose to configure my system for use as a software development toolbox. I practice scrum and use JIRA to manage all my software development projects. I currently run a self-contained JIRA instance from a guest virtual machine on OS X. This repo holds a process for provisioning a JIRA instance and all it's dependencies to a guest virtual machine.
